Link-API

A Simple Link Storage and Retrieval API - AKA Bookmark Keeping a curated set of your favorite links is a great way to keep track of interesting sites and content.

This simple sample provides a basic Links API option as well as a more advanced (but only mildly) option.

Use RAW link(s) for Creating Virtual APIs Basic Option - https://raw.githubusercontent.com/DevTestSolutions/Link-API/master/link-api.raml Advanced Option - https://raw.githubusercontent.com/DevTestSolutions/Link-API/master/link-api-advanced.raml

%RAML 0.8

title: link API baseUri: http://vapi.ca.com

/link: /{Title}: get: description: Returns links responses: 200: body: application/json: example: | { "title": {Title}, "user": {github_username}, "description": "Description", "url": "http://example.com" } post: description: posts links responses: 201: body: application/json: example: | { "title": {Title}, "user": {github_username}, "description": "Description", "url": "http://example.com" } Goto https://vai.ca.com to try the sample for yourself.
